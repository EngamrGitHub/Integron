# Next Company Site (White Background)

A clean, modern Next.js (App Router) website for a company with 3 divisions:
- Software Development
- 3D Design & Exhibition Stands
- Security Systems

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Customize
- Edit services in `lib/site-data.ts`
- Edit portfolio items in `lib/portfolio-data.ts`
- Update brand name in `components/Navbar.tsx` and `app/layout.tsx`
